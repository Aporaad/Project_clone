استخدم OSPF طبيعي للـ Return
router ospf 1
 network 0.0.0.0 255.255.255.255 area 0

لا تستخدم ECMP على Firewall (في هذا السيناريو)
set network virtual-router ospf ecmp enable no

اجعل نفس المسار ذهاب وعودة
set network virtual-router ospf protocol ospf area 0.0.0.0 interface ethernet1/5 metric 10
set network virtual-router ospf protocol ospf area 0.0.0.0 interface ethernet1/6 metric 100

set deviceconfig setting session tcp-reject-non-syn no

set deviceconfig setting session asymmetric-path bypass
