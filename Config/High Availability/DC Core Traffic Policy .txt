 frequency 5
ip sla schedule 2 life forever start-time now
track 2 ip sla 2 reachability

 match ip address ACL_VLAN-To-BRANCH
 set ip next-hop verify-availability 10.10.13.1 1 track 1
 set ip next-hop verify-availability 10.10.23.1 2 track 2
 set ip next-hop 10.10.34.2

route-map PBR_LAN-To-BRANCH permit 30
 set ip next-hop verify-availability 10.10.23.1 1 track 11
 set ip next-hop verify-availability 10.10.13.1 2 track 1
 set ip next-hop  10.10.34.2

 ip policy route-map PBR_VLAN-To-BRANCH
