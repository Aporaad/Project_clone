| **التوجيه والتحكم بالمسارات** | توجيه ثابت، بروتوكولات توجيه ديناميكية (OSPF/EIGRP)، التحكم بالمسارات (Route Maps). | توجيه ثابت، بروتوكولات توجيه ديناميكية (OSPF/BGP/RIP)، SD-WAN، سياسات التوجيه (PBR). | توجيه ثابت، بروتوكولات توجيه ديناميكية (OSPF/BGP/RIP)، SD-WAN، سياسات التوجيه (PBR). | توجيه ثابت، بروتوكولات توجيه ديناميكية (OSPF/BGP/RIP)، Policy-Based Forwarding (PBF). |
| **إعدادات الأمان والقواعد الأمنية** | قوائم التحكم بالوصول (ACLs)، مجموعات الكائنات، NAT (Static/Dynamic/PAT)، فحص الحالة (Stateful Inspection). | سياسات جدار الحماية، ملفات تعريف الأمان (AV, IPS, Web Filter, App Control, Email Filter)، NAT (Source/Destination). | قواعد جدار الحماية (User/Network-based)، ملفات تعريف الأمان (Web Protection, App Control, IPS, AV, Email Protection)، NAT (Source/Destination). | سياسات الأمان (App-ID, User-ID, Content-ID)، ملفات تعريف الأمان (Anti-Spyware, Vulnerability Protection, AV, URL Filtering, File Blocking, DLP)، NAT (Source/Destination). |
| **الفحص والتحقق** | فحص التطبيقات، IPS (مع Firepower)، تسجيل الأحداث (Syslog)، المراقبة (ASDM/SNMP). | Deep Packet Inspection (DPI)، تسجيل الأحداث (FortiAnalyzer/Syslog)، المراقبة والتقارير. | Deep Packet Inspection (DPI)، Synchronized Security، Sandstorm (Sandboxing)، تسجيل الأحداث (Sophos Central/Syslog)، المراقبة. | App-ID، Content-ID، WildFire (Sandboxing)، SSL Decryption، تسجيل الأحداث (Panorama/Syslog)، المراقبة. |
| **إدارة وفلترة المحتوى** | فلترة URL (دمج خارجي)، التحكم الأساسي بالتطبيقات، التحكم في النطاق الترددي (QoS). | تصفية الويب، التحكم في التطبيقات، منع فقدان البيانات (DLP)، جودة الخدمة (QoS). | تصفية الويب، التحكم في التطبيقات، منع فقدان البيانات (DLP)، جودة الخدمة (QoS). | URL Filtering، File Blocking، Data Filtering (DLP)، جودة الخدمة (QoS). |
